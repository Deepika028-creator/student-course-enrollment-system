# 🎓 Student Course Enrollment System

## 📌 Project Description
This is a MySQL mini project used to manage students, courses, enrollments, exams, and results.

## 🗂️ Database Tables
- Students
- Courses
- Enrollment
- Exams
- Results

## 🔑 Features
- List all students
- Show all courses
- Students enrolled in Database Systems
- Count total students per course
- Display exam marks with student and course
- Highest scoring student
- Students not enrolled in any course
- Courses with no exams conducted
- Average marks per course
- Students with grade 'A'
- Exams conducted in February
- Convert marks into grades
- Student with most enrollments

## 🛠️ Technologies Used
- MySQL

## ▶️ How to Run
1. Open MySQL
2. Copy and run the SQL file
3. Execute queries

## 📊 ER Concept
Students ↔ Enrollment ↔ Courses  
Students → Exams → Results
